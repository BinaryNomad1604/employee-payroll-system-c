# Employee Payroll System in C

A simple C program for managing employee payrolls. This project demonstrates basic file handling, data structures, and console-based user interface in C.

## Features

- Add new employee records  
- Display employee details  
- Store employee data in a file (`employee.dat`)  
- Simple console-based menu system
